## Getting Started
install libs
```bash
npm install
```

run the fake api:

```bash
json-server --watch products.json --port 3001
```

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000/products) with your browser to see the result.
